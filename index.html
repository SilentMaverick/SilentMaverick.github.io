RoboWar Combat Robot — Techfest IIT Bombay Arduino · C/C++ · Hardware
↳ Built & competed at Techfest 2024, IIT Bombay — Asia's largest science & tech festival

Designed and built a full combat robot from scratch for the RoboWar event at IIT Bombay Techfest — one of the most competitive robotics events in India with teams from IITs, NITs, and international colleges. Engineered the chassis, weapon system, and drive train, then programmed the entire control system on Arduino with custom PWM motor control, radio receiver integration, and real-time response tuning under millisecond latency constraints. Survived repeated weapon impacts in the arena — the robot worked exactly as designed under actual battle conditions.

Arduino Obstacle-Avoidance Rover Arduino · Ultrasonic · Motor Control
↳ Autonomous navigation using HC-SR04 sensors + L298N motor driver

Built an autonomous rover that maps its environment in real-time using ultrasonic distance sensors, makes directional decisions with a custom decision-tree algorithm, and steers via differential drive. Programmed entirely in C on Arduino — no libraries for the core logic, just raw sensor reads, timed pulses, and motor PWM. Tuned PID-like responsiveness through empirical testing across 50+ obstacle configurations.

Arduino Line-Following Robot Arduino · IR Sensors · PID Control
↳ Sub-50ms sensor-to-motor feedback loop on bare metal C

Built a line-following robot with IR sensor array and a hand-tuned proportional control loop achieving sub-50ms response times. The challenge wasn't just building it — it was understanding why it wobbled, tuning gain constants through first-principles reasoning, and eliminating signal noise from ambient light. Became my introduction to feedback control systems and real-time embedded constraints.

ZenoSQL — SQL Engine Written from Scratch in C C · Systems · Compilers
↳ github.com/SilentMaverick/ZenoSQL ★1

Built a working relational database engine in pure C — no libraries, no frameworks. Wrote a tokenizer, a recursive-descent parser for SQL syntax, a binary file storage layer, and a query executor from scratch. This isn't a tutorial project — it handles CREATE TABLE, INSERT, and SELECT on real persistent data. Doing this taught me more about how computers actually store and retrieve information than any course ever has.

Git Curator — GitHub Intelligence Tool JavaScript · GitHub API · UI
↳ github.com/SilentMaverick/Git_Curator ★2

Built a tool to search, track, sort, and analyze GitHub repos and users with AI-assisted discovery. Integrated the GitHub REST API with rate-limit handling, multi-criteria sorting, and live pagination over 50M+ repositories. I built this because I was genuinely frustrated with GitHub's search — the most hacker reason to build anything.

KeyMapper — System-Level Keyboard Remapping Engine C++ · OS Internals · Input Systems
↳ github.com/SilentMaverick

Wrote a keyboard remapping engine in C++ that intercepts key events at the OS level — raw scan codes, not OS key events. Implemented a config-file parser for binding definitions, support for chord mappings and modifier combos, and sub-millisecond rerouting latency. This forced me deep into how input devices, kernel drivers, and the OS input stack actually communicate.

GIF → ASCII Real-Time Renderer HTML · Canvas API · Pixel Sampling
↳ github.com/SilentMaverick/Gif_To_Character ★1

Converts animated GIFs to real-time ASCII art in the browser using the Canvas API and grayscale pixel-sampling. Processes 24+ FPS animations entirely client-side with no server — just math and pixels. Built it in an afternoon because I thought it would look cool. It does.
